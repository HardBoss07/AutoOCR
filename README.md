**AutoOCR** is a lightweight, distraction-free utility for Windows that instantly extracts text from images in your clipboard. It lives in your system tray and stays out of your way until you need it.

1. **Capture**: Take a screenshot (e.g., `Win + Shift + S`) or copy any image to your clipboard.
2. **Extract**: Press **`Shift + Alt + O`**.
3. **Paste**: The text is now in your clipboard! Just `Ctrl + V` anywhere.

* **System Tray Integration**: Runs silently in the background with a minimal footprint.
* **Multi-Language Support**: Supports English, German, Russian, Polish, and Hindi out of the box.
* **Start on Boot**: Automatically starts with Windows so it's always ready.
* **Fast & Local**: Powered by the Tesseract OCR engine. No data ever leaves your computer.

## ğŸŒ Adding More Languages

The app is designed to be expandable. If you need a language not included in the installer:

1. Download the `.traineddata` file for your language from the [Official Tesseract GitHub](https://github.com/tesseract-ocr/tessdata).
2. Open your installation folder (usually `C:\Program Files\AutoOCR\tessdata`).
3. Drop the file in. AutoOCR will detect it automatically!
